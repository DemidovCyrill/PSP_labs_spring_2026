# Лабораторная работа №3: Веб-приложение по палеонтологии

## Описание проекта

Данное веб-приложение представляет собой двухстраничный сайт, посвященный палеонтологии и динозаврам. Проект разработан в рамках лабораторной работы №3 по программированию сетевых приложений.

### Основные возможности:
- Просмотр списка динозавров на главной странице
- Детальная информация о каждом динозавре
- Интересные факты в модальных окнах
- Адаптивный дизайн для всех устройств

## Динозавры в приложении

| Динозавр | Эпоха | Интересный факт |
|----------|-------|-----------------|
| Тираннозавр Рекс | Поздний мел (68-66 млн лет) | Зубы достигали 30 см в длину |
| Трицератопс | Поздний мел (68-66 млн лет) | Паслись стадами, как современные бизоны |
| Стегозавр | Поздняя юра (155-145 млн лет) | Мозг размером с грецкий орех |

## Технологии

- **HTML5** — структура страниц
- **CSS3 + Bootstrap 5** — стилизация и адаптивность
- **JavaScript (ES6+)** — логика приложения
- **Модульная архитектура** — разделение на компоненты и страницы

## Запуск проекта

### Способ 1: Через Live Server (рекомендуется)

1. Установите расширение **Live Server** в VS Code
2. Откройте папку с проектом в VS Code
3. Нажмите правой кнопкой мыши на `index.html`
4. Выберите **"Open with Live Server"**

### Способ 2: Без установки Node.js

Проект использует Bootstrap CDN, поэтому не требует установки зависимостей. Просто откройте `index.html` в браузере.

## Функциональность

### Главная страница
- Отображение карточек всех динозавров
- Краткая информация (название, эпоха, короткое описание)
- Кнопка "Узнать больше" для перехода к деталям

### Страница динозавра
- Полная информация о динозавре:
  - Изображение
  - Название
  - Эпоха обитания
  - Рацион питания
  - Размеры (длина и вес)
  - Подробное описание
- Кнопка "Интересный факт" с модальным окном
- Кнопка "На главную" для навигации

### Модальное окно
- Отображает уникальные факты о выбранном динозавре
- Реализовано с использованием Bootstrap Modal
- Адаптивный дизайн
