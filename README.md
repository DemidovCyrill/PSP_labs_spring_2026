# Лабораторная работа №3: Веб-приложение по палеонтологии

## Описание проекта

Данное веб-приложение представляет собой двухстраничный сайт, посвященный палеонтологии и доисторическим животным. Проект разработан в рамках лабораторной работы №3 по программированию сетевых приложений.

### Основные возможности:
- Просмотр списка доисторических животных на главной странице
- Детальная информация о каждом животном
- Интересные факты в модальных окнах
- Адаптивный дизайн 
- Тёмная и светлая тема с сохранением в localStorage

## Технологии

- **HTML5** — семантическая структура страниц
- **CSS** — современные стили, переменные, анимации, адаптивность
- **JavaScript** — логика приложения, модульная архитектура
- **LocalStorage** — сохранение темы


## Запуск проекта

### Способ 1: Через Live Server (рекомендуется)

1. Установите расширение **Live Server** в VS Code
2. Откройте папку с проектом в VS Code
3. Нажмите правой кнопкой мыши на `index.html`
4. Выберите **"Open with Live Server"**

### Способ 2: Без установки Node.js

Проект использует Bootstrap CDN, поэтому не требует установки зависимостей. Просто откройте `index.html` в браузере.

## Функциональность

### Главная страница
- Отображение карточек всех доисторических животных
- Краткая информация (название, эпоха, короткое описание, рацион)
- Клик по карточке для перехода к детальной информации
- Тёмная/светлая тема с сохранением выбора
- Адаптивная сетка карточек

### Страница животного
- Полная информация:
  - Изображение
  - Название
  - Эпоха обитания
  - Рацион питания
  - Длина и вес
  - Подробное описание
- Кнопка "Интересный факт" с модальным окном
- Кнопка "Вернуться на главную" для навигации
- Единая тема с главной страницей

### Модальное окно
- Отображает уникальные факты о выбранном животном
- Реализовано с использованием кастомного модального окна (не Bootstrap)
- Плавные анимации открытия/закрытия

### Дополнительные возможности
- **Сохранение настроек**: Тема и цвет сохраняются в localStorage
