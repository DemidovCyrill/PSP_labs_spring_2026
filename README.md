# Лабораторная работа №5: Добавление AJAX-запросов к API

## Цель работы

Научиться взаимодействовать с внешним API через XMLHttpRequest (XHR). В ходе работы реализовано получение данных с сервера, фильтрация карточек по названию через query-параметр и выполнение DELETE-запросов для удаления ресурсов.

## План работы

1. Создать модуль Ajax для выполнения HTTP-запросов (GET, POST, PATCH, DELETE)
2. Создать модуль DinosaurUrls для централизованного хранения URL-адресов API
3. Перевести главную страницу на получение данных через API
4. Реализовать фильтрацию карточек по названию (query-параметр)
5. Перевести страницу динозавра на получение данных по ID через API
6. Добавить кнопку удаления динозавра с DELETE-запросом
7. Настроить CORS для работы с локальным сервером

## Описание проекта

ПалеоWise -- веб-приложение-энциклопедия доисторических животных. Состоит из клиентской части (HTML/CSS/JavaScript с модульной архитектурой) и серверной части (Node.js + Express, REST API). Клиент взаимодействует с сервером через асинхронные XMLHttpRequest-запросы без перезагрузки страницы.

### Основные возможности

- Загрузка списка динозавров с сервера через GET-запрос
- Поиск и фильтрация динозавров по названию
- Просмотр детальной информации о динозавре (загрузка по ID)
- Галерея изображений с полноэкранным режимом
- Удаление динозавра через DELETE-запрос
- Интерактивная 3D-модель на главной странице
- Страница функций для работы с массивами (лабораторная работа N4)
- Переключение темной и светлой темы

### Модуль Ajax (XMLHttpRequest)

Файл: `modules/ajax.js`

Класс Ajax предоставляет методы для выполнения HTTP-запросов к серверу через XMLHttpRequest.

```javascript
class Ajax {
    get(url, callback) {
        const xhr = new XMLHttpRequest();
        xhr.open('GET', url);
        xhr.send();
        xhr.onreadystatechange = () => {
            if (xhr.readyState === 4) {
                this._handleResponse(xhr, callback);
            }
        };
    }

    post(url, data, callback) {
        const xhr = new XMLHttpRequest();
        xhr.open('POST', url);
        xhr.setRequestHeader('Content-Type', 'application/json');
        xhr.send(JSON.stringify(data));
        xhr.onreadystatechange = () => {
            if (xhr.readyState === 4) {
                this._handleResponse(xhr, callback);
            }
        };
    }

    patch(url, data, callback) {
        const xhr = new XMLHttpRequest();
        xhr.open('PATCH', url);
        xhr.setRequestHeader('Content-Type', 'application/json');
        xhr.send(JSON.stringify(data));
        xhr.onreadystatechange = () => {
            if (xhr.readyState === 4) {
                this._handleResponse(xhr, callback);
            }
        };
    }

    delete(url, callback) {
        const xhr = new XMLHttpRequest();
        xhr.open('DELETE', url);
        xhr.send();
        xhr.onreadystatechange = () => {
            if (xhr.readyState === 4) {
                this._handleResponse(xhr, callback);
            }
        };
    }

    _handleResponse(xhr, callback) {
        try {
            const data = xhr.responseText ? JSON.parse(xhr.responseText) : null;
            callback(data, xhr.status);
        } catch (e) {
            console.error('Ошибка парсинга JSON:', e);
            callback(null, xhr.status);
        }
    }
}

export const ajax = new Ajax();
```

### Модуль URL-адресов API

Файл: `modules/urls.js`

Класс DinosaurUrls централизованно хранит все URL-адреса эндпоинтов API.

```javascript
class DinosaurUrls {
    constructor() {
        this.baseUrl = 'http://localhost:3000';
    }

    getDinosaurs() {
        return `${this.baseUrl}/dinosaurs/`;
    }

    getDinosaurById(id) {
        return `${this.baseUrl}/dinosaurs/${id}`;
    }

    getDinosaursByTitle(title) {
        return `${this.baseUrl}/dinosaurs/?title=${encodeURIComponent(title)}`;
    }

    deleteDinosaurById(id) {
        return `${this.baseUrl}/dinosaurs/${id}`;
    }
}

export const urls = new DinosaurUrls();
```

## Возможные ошибки и их решение

### Ошибка CORS
Access to XMLHttpRequest at 'http://localhost:3000/dinosaurs/' from origin
'http://127.0.0.1:5500' has been blocked by CORS policy

Решение: Установите и активируйте расширение CORS Unblock для Chrome.

### Ошибка подключения

GET http://localhost:3000/dinosaurs/ net::ERR_CONNECTION_REFUSED


Решение: Убедитесь, что сервер запущен командой npm run start в папке src.

### Карточки не отображаются

Решение: Проверьте консоль браузера (F12, вкладка Console) на наличие ошибок JavaScript. Убедитесь, что:
1. Сервер запущен и отвечает на http://localhost:3000/dinosaurs/
2. CORS Unblock активирован
3. В файле dinosaurs.json присутствуют данные с полями id, title, images
